Introduzione alle nostre API

Panoramica

Le API di UniFinder permettono agli sviluppatori di accedere ai nostri dati sulle università italiane e internazionali, corsi di laurea, recensioni e statistiche. Utilizziamo REST API con formato JSON per tutti gli endpoint.

Autenticazione

Per utilizzare le nostre API è necessario ottenere un API key. Le richieste autenticate devono includere l'header X-API-Key con il tuo token.

Rate Limiting

Le richieste sono limitate a 100 al minuto per API key. Gli header di risposta includeranno:

  • X-RateLimit-Limit: numero massimo di richieste
  • X-RateLimit-Remaining: richieste rimanenti nel periodo corrente
  • X-RateLimit-Reset: timestamp di reset del contatore
